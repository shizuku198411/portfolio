# Portfolio

## Session Attested
本リポジトリは、 [`SessionAttested`](https://github.com/shizuku198411/SessionAttested) による  
開発環境/プロセスの監査および署名を行っています。
これは、開発環境/プロセスにおいて、([監査ポリシー](attest/policy.yaml)) に定義されている  
exe/writerが検知されていないことを示すものです。  

- 監査サマリ: [`ATTESTED_SUMMARY`](ATTESTED_SUMMARY)
- 開発プロセスにおけるexe/writer一覧: [`ATTESTED_WORKSPACE_OBSERVED`](ATTESTED_WORKSPACE_OBSERVED)

詳細は [`SessionAttested`](https://github.com/shizuku198411/SessionAttested) を参照してください。
